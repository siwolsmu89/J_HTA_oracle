- 인라인 뷰(Inline View)
	- SQL문 안에서 정의된 뷰다.
	- SELECT문의 FROM절에 정의된 서브쿼리를 인라인 뷰라고 부른다.
	- 인라인 뷰는 SQL문이 실행되는 동안만 존재하고 즉시 사라진다. (데이터베이스 객체가 아니다)
	- 인라인 뷰 작성하기
	  -- 아래에서 FROM절의 () 안에 있는 것을 인라인 뷰라고 한다.
	  SELECT 컬럼명, 컬럼명, ...
	  FROM (SELECT 컬럼명, 컬럼명, ... 
		FROM 테이블명 
		[WHERE 조건식]) A, 테이블명 B
	  WHERE A.컬럼명 = B.컬럼명
	  * A는 서브쿼리의 실행결과 생성된 인라인 뷰의 별칭이다.
	  * FROM 절에 정의된 서브쿼리의 실행결과를 가상의 테이블로 취급한다.
	  * 테이블의 데이터를 가공한 것과 다른 테이블을 조인해야 하는 경우 많이 사용한다.
	  * 엄청 많이 사용된다.
